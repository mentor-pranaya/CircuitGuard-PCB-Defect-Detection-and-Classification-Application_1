Project Title: CircuitGuard-PCB Defect Detection and Classification Application

Internship Program: Infosys Springboard

Description:
This project aims to develop an automated defect detection and classification system for Printed Circuit Boards (PCBs) using image processing and deep learning techniques.

Development Platforms Used:
Google Colab: For implementing and training the machine learning and deep learning models.
Visual Studio Code (VS Code): For developing the frontend web application and user interface of the project.
GitHub: For version control and project submission.
